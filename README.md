# Pranathi | Portfolio

Welcome to my personal portfolio repository! This responsive and interactive website showcases my skills, projects, and educational journey. As a passionate Computer Science student specializing in Artificial Intelligence, I build modern, impactful applications that bridge creativity and technology.


## 🛠️ Built With

This portfolio is built directly using core web technologies to ensure fast performance, custom styling, and a clean codebase:

*   **HTML5** – Semantic structure and layout
*   **CSS3** – Custom styling, glassmorphism design, responsive layouts, and CSS variables
*   **JavaScript (Vanilla)** – Interactive elements, particle canvas animations, typing effects, and DOM manipulation
*   **AOS Library** – Animate On Scroll effects
*   **Font Awesome** – Iconography
*   **Google Fonts** – Typography (Inter & Space Grotesk)

## ✨ Features

*   **Interactive Hero Section**: Custom background particle canvas and dynamic typing effects.
*   **Fully Responsive Design**: Mobile-first architecture ensuring seamless viewing on desktop, tablet, and mobile.
*   **Smooth Navigation**: Sticky navbar with scroll-spy style active link tracking.
*   **Dynamic Portfolio Details**: Clean sections to display AI/Full-Stack projects (like *PathFinder Pro*).
*   **Visual Timelines & Skills**: Engaging graphical representation of technical proficiencies and educational background.
*   **Contact Form**: Working contact form integrated with Formspree.

## 💻 Running Locally

To view the portfolio locally on your machine:

1.  Clone this repository:
    ```bash
    git clone https://github.com/pranathiii2007-ops/portfolio.git
    ```
    *(Replace with your actual repository URL)*
2.  Navigate to the directory:
    ```bash
    cd portfolio
    ```
3.  Open `index.html` in your favorite web browser.
    *Tip: For the best development experience, use an extension like **Live Server** in VS Code.*
